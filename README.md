# Titanic Survival Prediction

Machine Learning model to predict passenger survival on Titanic.

## Results
- Accuracy: 72.7%
- Dataset: 1,046 passengers
- Model: Logistic Regression

## Key Findings
- Gender (sex) is the strongest predictor
- First class passengers had higher survival rate

## Technologies
- Python
- Scikit-learn

## How to run
`bash
python titanic.py
